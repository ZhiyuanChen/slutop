[![PyPI - Version](https://img.shields.io/pypi/v/slutop)](https://pypi.org/project/slutop)
[![PyPI - Python Version](https://img.shields.io/pypi/pyversions/slutop)](https://pypi.org/project/slutop)

`slutop` 是一个面向 [Slurm](https://slurm.schedmd.com) 集群的交互式监控工具。

它以持续刷新的方式，让你一目了然地查看整个集群的节点、分区、作业以及它们所消耗的资源。

## 安装

从 PyPI 安装最新发布版本：

```bash
pip install slutop
```

从源码安装最新开发版本：

```bash
pip install git+https://github.com/ZhiyuanChen/slutop
```

## 使用

```bash
slutop
```
