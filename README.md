`slutop` is an interactive monitor for [Slurm](https://slurm.schedmd.com) clusters.

It gives you an at-a-glance, continuously refreshing view of nodes, partitions, jobs, and the resources they consume across the cluster.

## Installation

Install the latest release from PyPI:

```bash
pip install slutop
```

Install the latest development version from source:

```bash
pip install git+https://github.com/ZhiyuanChen/slutop
```

## Usage

```bash
slutop
```
